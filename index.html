What is a Part-Based CMMS

A Computerized Maintenance Management Systems (CMMS) is used to help manage organizations asset maintenance needs.

Maintenance management is the tracking and handling of a company’s assets and involves upkeep, renewal, and replacement of damaged parts, machinery. ​ Our part-based maintenance software allows for more fine-tuned maintenance since we prioritize both the health of an asset as well as its parts.

Our Modules

Maintenance tasks are fulfilled by the following modules of our system

User Management

Users and their abilities are managed in this module. User's abilities are assigned with the help of cutsom user groups.

Physical Management

This module deals with the assets and parts in the system. Physical assets and parts are created from the logical parts, and logical assets.

Readmore

Work Order

Work orders are maintenance tasks created for an asset and need to be completed. This module deals with different types of work order and conducting the breakdown-analysis.

Readmore

Operational Calendar

This module allows us to enter the predicted running days of a locations and all assets in that location and help with the scheduling of maintenance.

Readmore

Preventative Maintenance

Work orders can be scheduled in advance using this module. A PM planner can create a work order template for assets that can be scheduled periodically, and pre-assigned to a technician.

Conditional Based Maintenance

This module allows for work orders to be created automatically when certain indicators measured through sensors show signs of upcoming failure.
